# Example Notebooks

This directory contains Jupyter notebooks demonstrating various use cases.

## Notebooks

### kg2neo4j.ipynb
Demonstrates how to load knowledge graphs and visualize them in Neo4j.

### abstract2KG.ipynb
Interactive notebook for processing single PubTator documents.

### QA_test.ipynb
Testing and quality assurance examples.

### creect_abstract.ipynb
Abstract correction and preprocessing utilities.

## Usage

```bash
jupyter notebook examples/notebooks/
```

## Requirements

```bash
pip install jupyter ipykernel
```
